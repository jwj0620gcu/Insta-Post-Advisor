개인정보처리방침

NoteRx (noterx.muran.tech) — 최종 업데이트: 2026년 4월

정보 수집

제출하신 게시글 스크린샷 및 텍스트 콘텐츠는 실시간 AI 분석에만 사용되며, 분석 완료 후 서버에 저장되지 않습니다. 귀하의 게시글 내용을 모델 학습에 사용하거나 제3자에게 판매하지 않습니다.

로컬 저장소

진단 기록은 귀하의 브라우저 IndexedDB에 저장되며, 완전히 로컬에서 관리됩니다. 브라우저 설정에서 언제든지 이 데이터를 삭제할 수 있으며, 서버는 귀하의 로컬 저장소에 접근할 수 없습니다.

익명 데이터

제품 경험 개선을 위해 다음과 같은 익명 데이터를 수집합니다:

  • IP 주소 — 접속 출처 구분에 사용되며, 개인 신원과 연결되지 않습니다
  • 진단 카테고리 — 사용자 니즈 분포 파악에 활용됩니다
  • 토큰 사용량 및 진단 소요 시간 — 성능 모니터링 및 비용 최적화에 활용됩니다
위 데이터는 모두 익명 형태로 기록되며, 특정 개인을 추적할 수 없습니다. 이 데이터를 광고 집행이나 개인 프로파일링에 사용하지 않겠습니다.

제3자 서비스

본 플랫폼은 AI 분석을 위해 MiMo API를 사용합니다. 진단 과정에서 귀하의 게시글 내용이 MiMo API로 전송되어 처리되며, 관련 데이터는 해당 개인정보처리방침의 적용을 받습니다. 그 외에는 귀하의 데이터를 다른 제3자와 공유하지 않습니다.

쿠키

본 플랫폼은 쿠키를 사용하지 않습니다. 귀하의 브라우징 행동을 추적하지 않으며, 어떠한 제3자 추적 도구도 사용하지 않습니다.

미성년자

NoteRx는 미성년자 팀(PageOne)이 개발하였으며, 모든 연령대의 사용자를 환영합니다. 본 플랫폼은 회원 가입을 요구하지 않으며, 개인 신원 정보를 수집하지 않습니다.
