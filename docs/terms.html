서비스 소개

Insta-Advisor는 5개의 다중 에이전트 토론 아키텍처 기반의 인스타그램 게시물 진단 플랫폼입니다. 사용자가 게시글(텍스트, 이미지, 스크린샷 등)을 제출하면 여러 에이전트가 콘텐츠 품질, 시각적 표현, 성장 잠재력, 사용자 반응 등 다양한 차원에서 분석하여 종합 진단 보고서를 도출합니다.

본 플랫폼은 한국 인스타그램 크리에이터에게 데이터 기반의 참고 의견을 제공하여 게시물 품질 향상을 돕는 것을 목적으로 합니다. 가천대학교 졸업 프로젝트로 개발된 오픈소스 서비스(MIT License)입니다.

사용자 책임

  • 업로드하는 게시물 콘텐츠는 합법적이어야 하며, 법령을 위반하는 내용을 포함해서는 안 됩니다.
  • 타인의 지식재산권, 개인정보 또는 기타 합법적 권리를 침해하는 콘텐츠를 업로드해서는 안 됩니다.
  • 사용자는 제출한 콘텐츠에 대해 전적인 책임을 집니다.

면책 조항

본 플랫폼이 제공하는 모든 진단 보고서, 점수 및 제안은 AI가 자동 생성한 것으로, 참고 목적으로만 제공되며 어떠한 운영 보증이나 효과 보장을 구성하지 않습니다. 실제 게시물 성과는 플랫폼 알고리즘, 시의성, 팔로워 특성 등 다양한 요소에 의해 영향을 받으므로, 본인의 판단과 병행하여 합리적으로 활용하시기 바랍니다.

데이터 안내

제출된 게시물 콘텐츠는 실시간 분석에만 사용되며, 분석 완료 후 서버에 저장되지 않습니다. 서버는 어떠한 개인 식별 정보도 저장하지 않으며, 콘텐츠를 모델 학습이나 제3자 공유 목적으로 사용하지 않습니다.

익명 사용 데이터

서비스 개선을 위해 다음과 같은 익명 데이터를 기록합니다:

  • IP 주소 (개인 신원과 연결되지 않음)
  • 진단 카테고리 및 토큰 사용량
  • 진단 소요 시간

위 데이터는 통계 분석 및 서비스 최적화 목적으로만 사용되며, 개인 신원 확인에 사용되지 않습니다.

지식재산권
